# Репозиторий приложений

Репозиторий содержит манифесты Kubernetes для развертывания тестового приложения podinfo через механизм синхронизации FluxCD.

## Предназначен для:
1) Хранения конфигурации приложения
2) Автоматического развертывания и обновления через синхронизацию с кластером
3) Версионирования изменений конфигурации приложения
